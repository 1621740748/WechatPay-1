
## 关于项目

本项目只是实现的基本的网站微信扫码支付、退款以及回调功能

在GlobalConfig文件中填写好微信支付相关的配置信息

更多说明可以看：[Java 网站微信扫码支付功能实现以及回调教程][1]

如果项目能够解决部分问题，厚颜要个 **Star**
